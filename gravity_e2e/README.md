# Gravity E2E Framework

This directory contains the End-to-End (E2E) testing framework for Gravity. It orchestrates local clusters and runs integration tests using Pytest.

## Key Documentation

-   **[User Manual](./docs/MANUAL.md)**: Main guide for running and configuring tests.
-   **[Adding Test Cases](./docs/adding_test_cases_guide.md)**: Developer guide for writing new test suites.

## Quick Start

Run the `single_node` test suite:

```bash
./run_test.sh single_node
```

See `MANUAL.md` for advanced usage.
